# 청류 개인정보처리방침

**최종 수정일:** 2026년 4월 25일

## 1. 개요

「청류」(이하 "본 확장 프로그램")는 웹 페이지에 표시된 대한민국 표준어 텍스트를 조선어(문화어) 표기법 및 어휘로 실시간 변환하는 Chrome 브라우저 확장 프로그램입니다.

본 개인정보처리방침은 본 확장 프로그램이 이용자의 정보를 어떻게 취급하는지 안내하기 위해 작성되었습니다.

## 2. 수집하는 개인정보

**본 확장 프로그램은 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.**

구체적으로 다음 정보를 수집하지 않습니다:

- 개인 식별 정보 (이름, 이메일, 주소, 전화번호 등)
- 웹 브라우징 기록 및 방문 페이지 URL
- 웹 페이지의 본문 내용
- 쿠키, 인증 토큰 등 인증 정보
- IP 주소, 기기 정보, 위치 정보
- 입력한 텍스트 또는 검색어
- 기타 일체의 사용자 활동 데이터

## 3. 데이터 처리 방식

본 확장 프로그램의 모든 텍스트 변환 작업은 **이용자의 브라우저 내부(클라이언트 사이드)** 에서만 이루어집니다. 웹 페이지의 텍스트는 외부 서버로 전송되지 않으며, 변환 과정에서 생성되는 데이터 역시 브라우저 밖으로 유출되지 않습니다.

### 로컬 저장소(chrome.storage.local) 사용

본 확장 프로그램은 Chrome의 로컬 저장소 API를 사용하여 다음 설정값만을 이용자의 기기에 저장합니다:

- 확장 프로그램 활성화/비활성화 상태
- 변환 옵션 설정값 (어휘, 두음법칙, 외래어, 띄어쓰기 등 기능별 ON/OFF)
- 사용자 정의 단어장
- 제외 사이트 목록

이 데이터는 이용자의 기기에만 존재하며, 외부로 전송되지 않습니다.

### 모델 캐시(IndexedDB) 사용

형태소 분석 기능에 사용되는 Kiwi 모델 파일은 최초 1회 다운로드된 후 이용자의 기기 내 IndexedDB에 캐시되며, 외부로 전송되지 않습니다. 캐시된 모델은 이후 오프라인 상태에서도 사용 가능합니다.

## 4. 권한 사용 목적

본 확장 프로그램이 요청하는 브라우저 권한과 그 사용 목적은 다음과 같습니다:

| 권한 | 사용 목적 |
|------|----------|
| `storage` | 이용자의 설정값을 로컬에 저장 및 유지 |
| `activeTab` | 현재 활성 탭의 DOM에 접근하여 텍스트 변환 스크립트 실행 |
| `offscreen` | Kiwi 형태소 분석기(WebAssembly)를 오프스크린 문서에서 실행 |
| `scripting` | 콘텐츠 스크립트와의 메시지 통신이 작동하지 않는 페이지에서도 설정 패널을 안정적으로 띄우거나 닫기 위해 해당 함수를 현재 탭에 직접 주입 |
| `host_permissions`<br>(`github.com`, `release-assets.githubusercontent.com`, `objects.githubusercontent.com`) | Kiwi 형태소 분석기 공개 모델 파일을 GitHub Releases에서 다운로드 |

콘텐츠 스크립트(content scripts)는 `<all_urls>` 패턴으로 모든 웹 페이지에서 실행되며, 이는 오직 페이지 내 텍스트를 조선어 규범에 맞게 변환하기 위한 단일 목적으로만 사용됩니다.

모든 권한은 텍스트 변환 및 그에 부수되는 기능을 위해서만 사용되며, 그 외의 용도로 사용되지 않습니다.

## 5. 제3자 제공

본 확장 프로그램은 이용자의 데이터를 제3자에게 판매, 전송, 공유하지 않습니다.

## 6. 외부 서버 통신

본 확장 프로그램은 이용자의 웹 브라우징 데이터, 페이지 내용, 입력 텍스트, 식별 정보 등을 **외부 서버로 전송하지 않습니다.**

다만, 형태소 분석 기능 제공을 위해 다음과 같은 **단방향 파일 다운로드 통신**이 발생할 수 있습니다:

- **Kiwi 형태소 분석기 공개 모델 다운로드**
  - **출처:** `https://github.com/bab2min/Kiwi/releases/`
  - **시점:** 확장 프로그램 최초 사용 시 또는 모델 버전 변경 시 (1회)
  - **내용:** Kiwi 형태소 분석기 공개 모델 파일(`.tgz`)
  - 이는 공개된 정적 파일을 단순히 다운로드하는 행위이며, 이용자의 개인정보·브라우징 데이터·입력 텍스트 등은 전송되지 않습니다.

다운로드된 모델은 이용자의 기기 내 IndexedDB에 캐시되어 이후에는 오프라인으로 동작합니다.

## 7. 아동 개인정보 보호

본 확장 프로그램은 아동을 포함한 누구의 개인정보도 수집하지 않으므로, 아동 개인정보 보호에 관한 별도의 조치가 필요하지 않습니다.

## 8. 개인정보처리방침의 변경

본 방침이 변경될 경우, 변경 사항은 본 페이지를 통해 공개됩니다. 중대한 변경이 있을 경우 확장 프로그램 업데이트 시 안내합니다.
